 Random Password Generator
 Автор Губайдуллин Никита


 Описание

Простое приложение для генерации случайных паролей с графическим интерфейсом. Позволяет настраивать длину и состав пароля (цифры, буквы, спецсимволы), сохраняет историю последних паролей в файл JSON.

 Требования

- Python 3.x.
- Библиотека `tkinter` (входит в стандартную поставку).

 Запуск

1. Установите Python (если не установлен).
2. Скачайте или клонируйте репозиторий.
3. Запустите:
   ```bash
   python main.py
   ```
4. Настройте параметры и сгенерируйте пароль.
5. История сохраняется автоматически.

 Возможные тесты

- Генерация пароля минимальной длины (4).
- Генерация пароля максимальной длины (32).
- Генерация только из цифр.
- Генерация только из букв.
- Генерация только из спецсимволов.
- Проверка сохранения и загрузки истории после перезапуска.
